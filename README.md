#  PhotoCloud

Rede social de fotos desenvolvida utilizando Docker, Docker Compose, Node.js e PostgreSQL.

---

#  Tecnologias Utilizadas

- Node.js
- Express
- PostgreSQL
- Docker
- Docker Compose

---

#  Arquitetura

Frontend → Node.js → PostgreSQL

---

#  Funcionalidades

- Publicação de fotos
- Exclusão de publicações
- Curtidas
- Comentários
- Persistência de dados
- Interface responsiva

---

#  Como Executar

```bash
docker compose up --build
